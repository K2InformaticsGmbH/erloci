Setting up Erloci on a new Windows 10 Installation

This manual describes how to build erloci and install the required dependencies, such as rebar3 and the Visual Studio Build Tools. It assumes a fresh Windows 10 installation.

Updating Windows
Mac Users: install Bootcamp Drivers
If Windows runs on a Mac, the Bootcamp Drivers should be installed so hardware such as trackpads and wireless cards work properly. It doesn't matter whether the Bootcamp Assistant was used to install Windows.
Virtual Box: install Guest Additions
If Windows runs in a virtual machine, the Guest Additions should be installed in order to allow features such as drag/dropping between the host and guest system.
Update Windows
Search → "Windows Update" to install updates. This takes a while and might require multiple restarts. Repeat until no updates are left.
Recommended Tools
Install Chrome
Click here to download Chrome (or Firefox, or a different browser of choice). Per default Edge is installed on Windows 10 with Bing as search engine. Google seems to deliver better results when searching for more software online later.
Download and Install Notepad++:
Click here, download "Notepad++ Installer 64-bit x64"
Download and install Git
Click here to download Git. This may take a while. Recommended install options are:
  • Use Notepad++ as Git's default editor (for people who don't know how to use vim)
  • Use Git from Git Bash only
  • Use MinTTY
Windows Options
Show file extensions
Search → "File Explorer Options" → Tab "View" → Uncheck "Hide Extensions for known file types"
Install Prerequisites
Install OCI
  • Make Folder "C:\Oracle"
  • Open Install\Oracle_12.2.0.1.0 (or go here and download the respective files)
  • Unzip all four .zip files to "C:\Oracle\instantclient_12_2"
Install Visual Studio Build Tools
It is possible to install all of Visual Studio instead of just the build tools. However, in this case the environment variables set later will ahve to be adjusted accordingly beause the paths are different. This can be circumvented by installing both Visual Studio and the Build Tools. The following instructions install just the Build Tools:
  • Click here, scroll down to "Tools for Visual Studio 2017" → "Build Tools f√ºr Visual Studio 2017" and download the downloader.
  • Start the installer, select the following workloads:
    • Visual C++ build tools
    • .NET desktop build tools
    • Universal Windows Platform build tools
  • As well as the following components:
    • C++/CLI support
Erlang
Go here, download "OTP 21.2 Windows 64-bit Binary File" and install it. Note: don't use newer version of Erlang with Erloci, they're not fully compatible (they may still be downloaded and installed, though).
